# UI/UX デザイン改善計画 (Soft Modern / Glassmorphism)

案2の「Soft Modern / Glassmorphism」に基づき、システムのビジュアルアイデンティティを刷新します。

## デザインコンセプト: "Luminous Clarity"
「透明感」と「光」をテーマに、複雑な受注管理業務を軽やかに、かつ直感的に行えるインターフェースを目指します。

### 1. カラーパレット & タイポグラフィ
- **Primary**: `Indigo 500` (#6366f1) - アクションと強調
- **Background**: `Slate 50` への淡い `Indigo/Rose` グラデーション
- **Surface**: 半透明の `White` (backdrop-blur)
- **Typography**: 
    - Display: `Outfit` (モダンで丸みのあるサンセリフ)
    - Body: `Plus Jakarta Sans` (視認性が高く洗練されたサンセリフ)

### 2. コンポーネントの指針
- **Cards**: 境界線を極限まで薄くし、`backdrop-blur-md` と控えめな `shadow-xl` で浮遊感を演出。
- **Buttons**: 微細なグラデーションと、ホバー時の「光る」ようなエフェクト。
- **Navigation**: サイドバー方式に変更し、画面の広がりを強調。

### 3. 改修ステップ
1. **`base.html` の刷新**: 全体レイアウトとフォント、共通スタイルの導入。
2. **デザインシステムの定義**: CSS変数を使用したカラー・スペーシングの固定。
3. **各画面の移行**: ダッシュボード -> 受注管理 -> マスタ管理の順で適用。
4. **磨き上げ**: ローディングアニメーションや、ステータス変更時のスムースな遷移。
